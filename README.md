# Kugoo 
Author: Isteh

Stack: 
- React
- Next.js
- TypeScript
- Sass (scss) modules
- Swiper
- Zustand

Features: 
- Three static (home/catalog/cart) pages and dynamic routes for every product
- Cart:
   - state cart save on localStorage
   - products have quantity
   - identic product stacked

How to view:
- clone / downlod repo
- npm i
- npm run build
- npm start

or go to https://kugoo-website.vercel.app/
